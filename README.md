## 👥 团队简介

智慧交通管理与技术应用团队致力于城市交通系统的智能化研究与技术创新，聚焦于交通规划、智能控制、车路协同等前沿领域。

**团队负责人**: chenshi_edu@gmail.com

---

## 🔬 研究方向

我们的研究涵盖以下四个核心方向：

### 1. 土地利用与交通一体化
- 城市土地利用与交通需求关系建模
- TOD（交通导向开发）规划理论与方法
- 城市空间结构优化
- 可持续城市交通系统设计

### 2. 面向车路云一体化的交通管控教学与实验平台
- 车路协同控制算法研究
- V2X通信协议实现
- 云端智能决策平台
- 智能网联交通实验教学系统

### 3. 大模型驱动的信号优化控制
- 基于大语言模型的交通信号控制
- 强化学习与LLM融合技术
- 多智能体协同优化
- 实时自适应信号控制系统

### 4. 硬件在环与嵌入式小车开发
- 硬件在环（HIL）测试平台
- 嵌入式控制系统开发
- 智能网联汽车原型设计
- 传感器融合与自主定位技术

## 📂 项目结构

本仓库采用 **Monorepo + Git Submodules** 架构，各研究方向独立管理：

```
smart-traffic-management/
├── docs/                    # 团队文档和研究资料
├── projects/                # 研究项目（Git Submodules）
│   ├── land-use-transport/          # 🔒 土地利用与交通一体化
│   ├── vehicle-road-cloud/          # 🔒 车路云一体化平台
│   ├── llm-signal-optimization/     # 🔒 大模型信号优化
│   └── hardware-in-loop/            # 🔒 硬件在环小车
├── scripts/                 # 管理脚本
├── resources/               # 共享资源
│   ├── papers/             # 参考文献
│   ├── datasets/           # 公开数据集
│   └── tools/              # 通用工具
└── README.md
```

> 🔒 **注意**: 各研究项目为私有仓库，仅对团队成员开放。

## 🚀 快速开始

### 团队成员

```bash
# 1. 克隆完整项目（包含所有子项目）
git clone --recursive https://github.com/YOUR_ORG/smart-traffic-management.git
cd smart-traffic-management

# 2. 运行初始化脚本
./scripts/setup-all.sh

# 3. 进入特定研究项目
cd projects/llm-signal-optimization
```
