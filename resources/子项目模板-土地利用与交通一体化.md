# 土地利用与交通一体化研究

> Land Use and Transport Integration Research

## 🎯 项目简介

本项目研究城市土地利用与交通系统的协同优化，探索可持续的城市发展模式。

**所属团队**: 智慧交通管理与技术应用团队  
**项目负责人**: 待补充  
**联系方式**: chenshi_edu@gmail.com

---

## 🔬 研究内容

### 1. 土地利用-交通关系建模
- 土地利用对出行需求的影响机制
- 交通可达性对土地价值的作用
- 职住平衡与通勤模式研究
- 空间句法与城市形态分析

### 2. TOD理论与应用
- 交通导向开发（TOD）模式研究
- 轨道站点周边土地利用优化
- TOD效益评估方法
- 可持续城市发展策略

### 3. 综合规划方法
- 土地利用-交通集成模型
- 多情景规划与评估
- 政策仿真与决策支持
- 大数据驱动的规划分析

## 🔧 使用示例

### 示例1：提取OSM路网数据

```python
import osmnx as ox
import geopandas as gpd

# 下载北京市路网
G = ox.graph_from_place("Beijing, China", network_type='drive')

# 转换为GeoDataFrame
nodes, edges = ox.graph_to_gdfs(G)

# 保存
edges.to_file("data/processed/beijing_road_network.shp")
```
