# 大模型驱动的信号优化控制

> LLM-Driven Traffic Signal Optimization and Control

## 🚀 快速开始

### 环境要求

**必需**:
- Python 3.9+
- PyTorch 2.0+
- SUMO 1.10+ (交通仿真)
- CUDA 11.8+ (GPU训练)

**可选**:
- Weights & Biases (实验跟踪)
- TensorBoard (可视化)

### 安装

```bash
# 克隆项目
git clone https://github.com/YOUR_ORG/llm-signal-optimization.git
cd llm-signal-optimization

# 创建conda环境（推荐）
conda create -n llm-traffic python=3.9
conda activate llm-traffic

# 安装PyTorch（根据CUDA版本）
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# 安装其他依赖
pip install -r requirements.txt

# 安装SUMO
# Ubuntu:
sudo apt install sumo sumo-tools
# Windows: 从官网下载安装包
```

```bash

python examples/demo_pretrained.py

# 启动可视化仿真
python simulation/run_visualization.py --network grid --model hybrid
```

## 🧪 核心功能

### 1. LLM理解交通场景

```python
from src.models import LLMController

# 初始化LLM控制器
controller = LLMController(model_name="gpt-4")

# 输入交通场景描述
traffic_state = {
    "north_queue": 15,
    "south_queue": 8,
    "east_queue": 20,
    "west_queue": 5,
    "time_of_day": "rush_hour",
    "weather": "rainy"
}

# LLM推理最优策略
action = controller.predict(traffic_state)
print(f"建议配时: {action}")
```

### 2. 强化学习优化

```python
from src.models import RLAgent
from src.training import train_rl

# 创建RL智能体
agent = RLAgent(
    state_dim=20,
    action_dim=4,
    algorithm='PPO'
)

# 训练
train_rl(
    agent=agent,
    env='sumo-intersection',
    episodes=10000
)
```

### 3. LLM + RL 混合模型

```python
from src.models import HybridModel

# 创建混合模型
model = HybridModel(
    llm_backbone='claude-3',
    rl_algorithm='A3C'
)

# LLM提供高层策略，RL执行细粒度控制
strategy = model.llm_plan(traffic_state)
action = model.rl_execute(strategy, real_time_data)
```

## 🔧 训练自己的模型

### 步骤1: 准备数据

```bash
# 收集SUMO仿真数据
python data_processing/collect_data.py \
    --network networks/beijing_cbd.net.xml \
    --routes routes/morning_peak.rou.xml \
    --duration 7200 \
    --output data/training_data.pkl
```

### 步骤2: 训练RL基线

```bash
# 训练PPO智能体
python training/train_rl.py \
    --config configs/training_configs/ppo.yaml \
    --data data/training_data.pkl \
    --output models/rl_baseline.pth
```

### 步骤3: 微调LLM

```bash
# 使用LoRA微调LLM
python training/train_llm.py \
    --base_model meta-llama/Llama-2-7b \
    --method lora \
    --data data/traffic_instructions.json \
    --output models/llm_traffic.pth
```

### 步骤4: 训练混合模型

```bash
# 联合训练
python training/train_hybrid.py \
    --llm_checkpoint models/llm_traffic.pth \
    --rl_checkpoint models/rl_baseline.pth \
    --config configs/training_configs/hybrid.yaml
```

## 📖 配置说明

### LLM配置 (configs/model_configs/llm_config.yaml)

```yaml
llm:
  model_name: "claude-3-sonnet"
  temperature: 0.7
  max_tokens: 500
  system_prompt: |
    你是一个交通信号控制专家。根据当前交通状态，
    给出最优的信号配时策略。

reasoning:
  use_chain_of_thought: true
  few_shot_examples: 5
```

### RL配置 (configs/model_configs/rl_config.yaml)

```yaml
rl:
  algorithm: "PPO"
  network:
    hidden_layers: [256, 256, 128]
    activation: "relu"
  
  hyperparameters:
    learning_rate: 0.0003
    gamma: 0.99
    gae_lambda: 0.95
    clip_epsilon: 0.2
    
  training:
    batch_size: 64
    epochs: 10
    buffer_size: 100000
```
