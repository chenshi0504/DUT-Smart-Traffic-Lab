## 🚀 快速开始

### 环境要求

**软件环境**:
- Ubuntu 20.04 LTS (推荐，用于ROS)
- ROS Noetic
- MATLAB R2020a+ (HIL仿真)
- STM32CubeIDE / Arduino IDE
- Python 3.8+

**硬件需求**:
- 智能小车硬件平台（详见硬件清单）
- HIL仿真工作站（可选）
- 调试器 (ST-Link V2, J-Link等)

### 安装ROS环境

```bash
# 安装ROS Noetic (Ubuntu 20.04)
sudo apt update
sudo apt install ros-noetic-desktop-full

# 安装依赖
sudo apt install python3-rosdep python3-rosinstall python3-catkin-tools

# 初始化rosdep
sudo rosdep init
rosdep update

# 配置环境
echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc
source ~/.bashrc
```

### 编译ROS工作空间

```bash
# 创建工作空间
mkdir -p ~/catkin_ws/src
cd ~/catkin_ws/src

# 克隆本项目
git clone https://github.com/YOUR_ORG/hardware-in-loop-vehicle.git

# 安装依赖
cd ~/catkin_ws
rosdep install --from-paths src --ignore-src -r -y

# 编译
catkin_make

# 配置环境
source devel/setup.bash
```

### 快速测试

```bash
# 启动小车仿真
roslaunch vehicle_control simulation.launch

# 启动可视化 (另一个终端)
rviz -d config/vehicle.rviz

# 发送测试指令
rostopic pub /cmd_vel geometry_msgs/Twist "linear: {x: 0.5}" "angular: {z: 0.2}"
```

## 💻 嵌入式开发

### STM32固件开发

```c
// firmware/stm32/Core/Src/main.c

// 主控制循环
void Control_Loop(void) {
    // 读取传感器
    IMU_Data imu = IMU_Read();
    Encoder_Data encoder = Encoder_Read();
    
    // 接收上位机指令
    Control_Command cmd = UART_ReceiveCommand();
    
    // PID控制
    float left_speed = PID_Compute(&pid_left, cmd.left_target, encoder.left);
    float right_speed = PID_Compute(&pid_right, cmd.right_target, encoder.right);
    
    // 输出PWM
    Motor_SetSpeed(MOTOR_LEFT, left_speed);
    Motor_SetSpeed(MOTOR_RIGHT, right_speed);
    
    // 发送反馈
    UART_SendFeedback(imu, encoder);
}
```

### Raspberry Pi开发

```python
# firmware/raspberry_pi/main.py
import rospy
from sensor_msgs.msg import LaserScan, Image
from geometry_msgs.msg import Twist

class VehicleController:
    def __init__(self):
        # 订阅传感器话题
        rospy.Subscriber('/scan', LaserScan, self.lidar_callback)
        rospy.Subscriber('/camera/image', Image, self.camera_callback)
        
        # 发布控制指令
        self.cmd_pub = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
        
    def lidar_callback(self, msg):
        # 处理激光雷达数据
        obstacle_detected = self.detect_obstacle(msg.ranges)
        
        if obstacle_detected:
            self.emergency_stop()
            
    def camera_callback(self, msg):
        # 图像处理
        lane_center = self.detect_lane(msg)
        self.follow_lane(lane_center)
```

## 🎯 HIL仿真测试

### Simulink模型

1. **车辆动力学模型**
   - 文件: `hil-simulation/simulink-models/vehicle_dynamics.slx`
   - 功能: 模拟车辆运动

2. **传感器模型**
   - 文件: `hil-simulation/simulink-models/sensor_models.slx`
   - 功能: 模拟传感器输出（带噪声）

3. **环境模型**
   - 文件: `hil-simulation/simulink-models/environment.slx`
   - 功能: 道路、障碍物等

### 运行HIL测试

```bash
# 启动Simulink实时仿真
matlab -r "open('hil-simulation/simulink-models/hil_test.slx'); sim('hil_test')"

# 连接真实硬件
python tools/hil_interface.py --port /dev/ttyUSB0 --baudrate 115200

# 运行测试场景
python hil-simulation/test-scenarios/run_test.py --scenario obstacle_avoidance
```

## 🤖 ROS开发

### 启动小车

```bash
# 启动所有节点
roslaunch vehicle_control vehicle_bringup.launch

# 单独启动模块
rosrun sensor_fusion ekf_localization  # 定位
rosrun path_planning astar_planner     # 路径规划
rosrun vehicle_control pid_controller  # 控制
```

### 查看话题

```bash
# 列出所有话题
rostopic list

# 查看激光雷达数据
rostopic echo /scan

# 监控控制指令
rostopic echo /cmd_vel
```

### ROS节点架构

```
sensor_drivers
├── lidar_driver        → /scan
├── camera_driver       → /camera/image
├── imu_driver          → /imu/data
└── gps_driver          → /gps/fix

sensor_fusion
└── ekf_localization    → /odometry/filtered

perception
├── obstacle_detection  → /obstacles
└── lane_detection      → /lane_center

planning
├── path_planner        → /planned_path
└── behavior_planner    → /behavior

control
└── pid_controller      → /cmd_vel

hardware_interface
└── vehicle_interface   → (串口通信到STM32)
```

## 📊 数据采集与分析

### 记录rosbag

```bash
# 记录所有话题
rosbag record -a -O experiment_001.bag

# 记录特定话题
rosbag record /scan /camera/image /odom -O sensors.bag
```

### 分析数据

```python
# tools/data_analysis.py
import rosbag
import matplotlib.pyplot as plt

bag = rosbag.Bag('experiment_001.bag')

# 提取数据
timestamps = []
velocities = []

for topic, msg, t in bag.read_messages(topics=['/cmd_vel']):
    timestamps.append(t.to_sec())
    velocities.append(msg.linear.x)

# 可视化
plt.plot(timestamps, velocities)
plt.xlabel('Time (s)')
plt.ylabel('Velocity (m/s)')
plt.show()
```

## 🔧 调试与故障排除

### 常见问题

**Q: 激光雷达无法连接**
```bash
# 检查设备
ls /dev/ttyUSB*

# 添加权限
sudo chmod 666 /dev/ttyUSB0

# 测试连接
rosrun rplidar_ros rplidarNode
```

**Q: 电机不转**
- 检查电源连接
- 检查PWM信号
- 使用逻辑分析仪调试

**Q: IMU数据漂移**
- 重新校准传感器
- 检查滤波参数
- 静止时记录bias
