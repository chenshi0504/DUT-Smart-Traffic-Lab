## 🚀 快速开始

### 环境要求

**基础环境**:
- Python 3.8+
- Docker & Docker Compose
- Node.js 14+ (前端界面)

**仿真工具** (可选):
- SUMO 1.10+
- CARLA 0.9.13+

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/YOUR_ORG/vehicle-road-cloud-platform.git
cd vehicle-road-cloud-platform

# 使用Docker快速启动
docker-compose up -d

# 或手动安装
pip install -r requirements.txt
```

### 快速演示

```bash
# 启动云端服务
python src/cloud/main.py

# 启动仿真环境（另一个终端）
python simulation/run_simulation.py --scenario intersection

# 访问可视化界面
# http://localhost:8080
```

## 🎓 教学实验

### 实验1: 协同式自适应巡航控制 (CACC)

**实验目的**: 理解V2V通信在车辆编队中的应用

**实验步骤**:
1. 启动仿真环境
2. 部署多辆智能网联车辆
3. 配置CACC参数
4. 观察车辆间距保持效果

```bash
cd experiments/exp1_cacc
python run_cacc_demo.py
```

### 实验2: 智能交叉口协同通行

**实验目的**: 学习V2I通信和协同决策

**实验步骤**:
1. 配置交叉口场景
2. 部署RSU设备
3. 车辆接收信号灯信息
4. 优化通行效率

```bash
cd experiments/exp2_intersection
python run_intersection_demo.py
```

### 实验3: 应急车辆优先通行

**实验目的**: 实现基于V2X的应急响应

**实验步骤**:
1. 设置应急场景
2. 应急车辆广播优先请求
3. 其他车辆协同避让
4. 信号灯优先放行

## 🔧 开发指南

### 车端开发

```python
# src/vehicle/main.py
from vehicle.perception import PerceptionModule
from vehicle.communication import V2XModule

class VehicleAgent:
    def __init__(self):
        self.perception = PerceptionModule()
        self.v2x = V2XModule()
        
    def run(self):
        # 感知周围环境
        obstacles = self.perception.detect_obstacles()
        
        # 接收V2X消息
        messages = self.v2x.receive()
        
        # 决策与控制
        action = self.make_decision(obstacles, messages)
        self.execute(action)
```
